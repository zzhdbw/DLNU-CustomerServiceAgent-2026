# 小米14 规格参数

## 基本信息
- 品牌：小米（Xiaomi）
- 型号：小米14
- 上市时间：2023年10月
- 操作系统：Xiaomi HyperOS（基于Android 14）
- 颜色：黑色、白色、岩石青、雪山粉

## 处理器与内存
- CPU：第三代骁龙8移动平台（Snapdragon 8 Gen 3）
- GPU：Adreno 750
- RAM：8GB / 12GB / 16GB LPDDR5X
- 存储：256GB / 512GB / 1TB UFS 4.0

## 屏幕
- 尺寸：6.36英寸
- 类型：OLED 柔性直屏
- 分辨率：2670×1200
- 刷新率：120Hz LTPO
- 亮度：3000nit 峰值亮度
- 护眼：DC调光、莱茵低蓝光认证

## 摄像头
- 后置主摄：5000万像素（光影猎人900，F1.6，OIS光学防抖）
- 超广角：5000万像素（F2.2，115°）
- 长焦：5000万像素（75mm等效，OIS）
- 前置：3200万像素
- 视频录制：8K@24fps、4K@60fps

## 电池与充电
- 电池容量：4610mAh
- 有线充电：90W 小米澎湃快充
- 无线充电：50W 无线快充
- 无线反充：10W

## 网络与连接
- 5G：SA/NSA双模
- Wi-Fi：Wi-Fi 7（802.11be）
- 蓝牙：5.4
- NFC：支持
- 红外遥控：支持

## 其他特性
- 指纹：屏幕光学指纹
- 防护等级：IP68
- 重量：188g（玻璃版）/ 193g（素皮版）
- 厚度：8.20mm（玻璃版）/ 8.28mm（素皮版）

- 12GB+256GB：¥4299
- 16GB+512GB：¥4599
- 16GB+1TB：¥4999
