# Redmi K70 规格参数

## 基本信息
- 品牌：Redmi（红米）
- 型号：Redmi K70
- 上市时间：2023年11月
- 操作系统：Xiaomi HyperOS（基于Android 14）
- 颜色：晴雪白、墨羽黑、浅茗青、竹月蓝

## 处理器与内存
- CPU：第二代骁龙8移动平台（Snapdragon 8 Gen 2）
- GPU：Adreno 740
- RAM：12GB / 16GB LPDDR5X
- 存储：256GB / 512GB / 1TB UFS 4.0

## 屏幕
- 尺寸：6.67英寸
- 类型：OLED 柔性直屏
- 分辨率：3200×1440（2K）
- 刷新率：120Hz
- 亮度：4000nit 峰值亮度
- 护眼：3840Hz PWM高频调光（小米青山护眼方案）

## 摄像头
- 后置主摄：5000万像素（光影猎人800，F1.6，OIS光学防抖）
- 超广角：800万像素（F2.2）
- 微距：200万像素
- 前置：1600万像素
- 视频录制：4K@60fps

## 电池与充电
- 电池容量：5000mAh
- 有线充电：120W 澎湃快充
- 无线充电：不支持

## 网络与连接
- 5G：SA/NSA双模
- Wi-Fi：Wi-Fi 7
- 蓝牙：5.3
- NFC：支持
- 红外遥控：支持

## 其他特性
- 指纹：屏幕光学指纹
- 防护等级：IP54（生活防泼溅）
- 重量：209g
- 厚度：8.21mm
