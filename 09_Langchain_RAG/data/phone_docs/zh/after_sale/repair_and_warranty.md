# 维修与保修

## 保修政策详情

### 小米保修政策
- 主机（含电池）：1年质保
- 充电器、数据线：6个月质保
- 耳机：3个月质保
- 小米Care（延保服务）：延长1年，¥199-¥399（机型不同定价不同）
- 碎屏险：¥149-¥299/年，每年限2次免费换屏

### 华为保修政策
- 主机（含电池）：1年质保
- 充电器、数据线：6个月质保
- 华为Care+：原价+¥299起，含延保1年+2次意外保
- 碎屏服务宝：¥199-¥399/年，限2次换屏服务

### 以下情况不在保修范围
1. 人为损坏（摔碎、进水、挤压变形）
2. 未经授权拆修
3. 使用非官方配件造成的损坏
4. 不可抗力（水灾、火灾、雷击等）
5. 外观正常磨损（划痕、掉漆）

## 维修流程

### 小米维修流程
1. **线上预约**：小米商城APP→我的→服务→预约维修
2. **到店维修**：带上手机+购机凭证（电子发票即可）→前往小米之家/小米授权服务中心
3. **寄修服务**：小米商城APP申请寄修→快递上门取件→维修中心检测→维修后寄回
4. **维修时长**：简单问题（换电池、换屏）1-2小时；主板问题3-7天
5. **备用机**：部分门店可提供备用机

### 华为维修流程
1. **线上预约**：华为消费者业务服务官网/华为商城APP→服务→预约维修
2. **到店维修**：带上手机+购机凭证→前往华为客户服务中心
3. **寄修服务**：微信公众号"华为服务"申请→顺丰上门取件→寄回维修
4. **维修时长**：简单问题1-2小时；主板问题3-5天
5. **备用机**：部分门店提供备用机服务

## 维修建议
- 优先选择官方售后，第三方维修可能影响后续保修
- 维修前备份数据并退出个人账号（小米账号/华为账号）
- 保留维修单据，同一问题维修后3个月内复发可免费再次维修
- 超过保修期可咨询官方是否有"电池换新优惠活动"（通常半年一次）
