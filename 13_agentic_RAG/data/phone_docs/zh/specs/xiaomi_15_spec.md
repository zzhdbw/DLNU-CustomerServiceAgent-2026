# 小米15 规格参数

## 基本信息
- 品牌：小米（Xiaomi）
- 型号：小米15
- 上市时间：2024年10月
- 操作系统：Xiaomi HyperOS 2（基于Android 15）
- 常规颜色：黑色、白色、浅草绿、丁香紫
- 特别版本：亮银版、定制版（20款定制色彩+2种中框颜色）、钻石限定版

## 处理器与内存
- CPU：骁龙8至尊版移动平台（Snapdragon 8 Elite），3nm制程，双超大核+6颗性能大核架构，最高主频4.32GHz
- GPU：Adreno 830
- RAM：12GB / 16GB LPDDR5X
- 存储：256GB / 512GB / 1TB UFS 4.0

## 屏幕
- 尺寸：6.36英寸
- 类型：OLED 柔性直屏（CrystalRes AMOLED）
- 分辨率：2670×1200（1.5K），460ppi
- 刷新率：1-120Hz LTPO 可变刷新率，支持分区刷新
- 亮度：多场景峰值亮度3200nit，全局激发亮度1000nit
- 护眼：全程DC调光、TÜV莱茵低蓝光/无频闪/昼夜节律友善认证
- 其他特性：1.38mm超窄四等边边框（LIPO立体封装）、小米龙晶玻璃（耐摔性提升10倍）、湿手触控技术、Dolby Vision / HDR10+

## 摄像头
- 后置主摄：5000万像素（光影猎人900，1/1.3"，F1.62，OIS，23mm等效焦距）
- 超广角：5000万像素（三星JN1，F2.2，115°，14mm等效焦距）
- 长焦：5000万像素（三星JN5，F2.0，OIS，60mm等效焦距，支持浮动对焦）
- 前置：3200万像素（居中挖孔）
- 视频录制：8K@24/30fps、4K@24/30/60fps
- 影像技术：徕卡Summilux光学系统、Xiaomi AISP大模型计算摄影平台2.0

## 电池与充电
- 电池容量：5400mAh（小米金沙江电池，能量密度850Wh/L）
- 有线充电：90W 小米澎湃快充（配备澎湃P3快充芯片）
- 无线充电：50W 无线快充（配备澎湃G1电池管理芯片）
- 无线反充：支持
- 磁吸无线充：支持，需搭配磁吸配件（最高30W）

## 网络与连接
- 5G：全网通5G，SA/NSA双模
- Wi-Fi：Wi-Fi 7（802.11be）
- 蓝牙：5.4
- NFC：支持
- 红外遥控：支持
- 其他连接：小米星辰通信系统，支持小米星辰无网通信（3.5km无网络双向通话）

## 其他特性
- 指纹：超声波屏下指纹（支持湿手解锁）
- 防护等级：IP68
- 重量：191g（玻璃版）/ 192g（亮银版）/ 189g（钻石限定版）
- 厚度：8.08mm（玻璃版）/ 8.48mm（亮银版）/ 8.36mm（钻石限定版）
- 其他：一体航空铝边框、一体化3D冷雕火山口、四曲包裹式中框、X轴线性马达、立体声双扬声器、翼型环状冷却系统
