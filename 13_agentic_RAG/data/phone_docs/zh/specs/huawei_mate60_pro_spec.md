# 华为 Mate 60 Pro 规格参数

## 基本信息
- 品牌：华为（HUAWEI）
- 型号：Mate 60 Pro
- 上市时间：2023年9月
- 操作系统：HarmonyOS 4.0
- 颜色：雅丹黑、白沙银、南糯紫、雅川青

## 处理器与内存
- CPU：麒麟9000S
- GPU：Maleoon 910
- RAM：12GB LPDDR5
- 存储：256GB / 512GB / 1TB
- 扩展存储：支持NM存储卡（最大256GB）

## 屏幕
- 尺寸：6.82英寸
- 类型：OLED 等深四曲面屏
- 分辨率：2720×1260
- 刷新率：120Hz LTPO
- 亮度：3000nit 峰值亮度
- 护眼：1440Hz PWM调光

## 摄像头（XMAGE影像）
- 后置主摄：5000万像素（F1.4-F4.0可变光圈，OIS）
- 超广角：1200万像素（F2.2）
- 长焦：4800万像素（3.5倍光学变焦，OIS）
- 前置：1300万像素+3D深感摄像头
- 视频录制：4K@60fps

## 电池与充电
- 电池容量：5000mAh
- 有线充电：88W 有线超级快充
- 无线充电：50W 无线超级快充
- 无线反充：20W

## 网络与连接
- 网络制式：4G全网通（支持卫星通信）
- 卫星通信：支持天通卫星通话+北斗卫星消息
- Wi-Fi：Wi-Fi 6（802.11ax）
- 蓝牙：5.2
- NFC：支持
- 红外遥控：支持

## 其他特性
- 指纹：屏幕指纹
- 解锁方式：面部识别+屏幕指纹
- 防护等级：IP68
- 重量：225g
- 厚度：8.1mm
