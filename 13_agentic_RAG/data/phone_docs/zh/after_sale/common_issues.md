# 常见使用问题

## 系统与软件问题

### 手机卡顿怎么解决？
1. 清理后台应用：一键清理或重启手机
2. 清理缓存：设置→存储→清理缓存
3. 检查存储空间：剩余空间建议保持在10GB以上
4. 关闭不必要的动画效果：开发者选项中调低动画缩放
5. 如持续卡顿，尝试备份数据后恢复出厂设置
6. 若以上无效，可能硬件问题，建议前往售后检测

### 手机发烫严重怎么办？
正常使用微温是正常现象，如果发烫到明显烫手：
1. 移除手机壳辅助散热
2. 关闭高功耗应用（游戏、视频通话等）
3. 降低屏幕亮度和刷新率
4. 避免充电时玩大型游戏
5. 检查是否有后台异常耗电应用
6. 如果频繁过热保护提示，建议前往售后检测

### 系统更新后出现BUG怎么办？
1. 尝试重启手机，部分问题重启后可恢复
2. 进入设置检查是否有修复补丁更新
3. 双清缓存（Recovery模式清除cache分区）
4. 如严重影响使用，可退回到前一系统版本（小米支持7天内回退）
5. 通过官方社区/客服反馈BUG

### 升级HarmonyOS后数据会丢失吗？
正规OTA升级不会丢失数据，但建议升级前进行云备份或电脑备份。升级后部分第三方应用可能需重新适配，如果遇到闪退，在应用商店更新到最新版即可。

## 屏幕与显示问题

### 屏幕有坏点/亮点怎么办？
屏幕坏点在1-3个以内是行业标准允许范围。如果超过3个或严重影响使用，保修期内可免费换屏。小米/华为线下售后均可检测处理。

### 屏幕摔碎了怎么处理？
- 保修期内：人为损坏不在保修范围，需自费维修
- 建议前往官方售后换屏，避免第三方屏影响触控和显示效果
- 换屏价格参考：小米14约¥900、Redmi K70约¥550、Mate 60 Pro约¥1399、P60约¥1099
- 建议购买碎屏险（购机时¥149-¥299/年）

### 贴膜影响指纹识别吗？
钢化膜和UV膜可能影响光学指纹识别。建议使用官方原厂膜或超薄水凝膜。贴膜后如果指纹不灵敏，可重新录入指纹（同一手指录入2-3次以提高识别率）。

## 电池与充电问题

### 电池不耐用怎么办？
1. 检查电池健康度（设置→电池→电池健康）
2. 关闭5G网络、降低屏幕刷新率可延长续航
3. 移除后台自启动高耗电应用
4. 电池健康度低于80%建议更换电池
5. 换电池价格：小米14约¥159、Redmi K70约¥99、Mate 60 Pro约¥299

### 充电慢怎么排查？
1. 检查是否使用原装充电器和数据线
2. 充电口是否进灰/氧化（用牙签轻轻清理）
3. 边充边玩会降低充电速度
4. 电池温度过高时会触发充电保护降速
5. 尝试重启手机
6. 如果持续慢充，建议检测充电IC模块

### 可以边充电边玩手机吗？
可以，但不建议。边充边玩：1）发热加重，长期影响电池寿命；2）充电速度变慢；3）存在安全隐患（劣质充电器可能导致事故）。建议充电时休息一会儿。
