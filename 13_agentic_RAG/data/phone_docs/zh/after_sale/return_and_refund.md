# 退换货与退款

## 退货条件

### 7天无理由退货
- 适用：通过官方商城、京东、天猫等正规渠道购买的手机
- 要求：手机未激活、外观无划痕、配件齐全、包装完好
- **注意**：手机一旦激活（连接网络/插入SIM卡），默认已使用，不支持无理由退货
- 不适用：定制机、合约机、二手商品

### 15天质量问题换货
- 购机15天内，经检测确认为产品质量问题（非人为损坏），可免费更换同型号新机
- 需提供购机发票或电子凭证
- 检测流程：前往官方售后网点→工程师检测→出具检测报告→凭报告换新机
- 换机后质保期重新计算

### 三包期内维修
超过15天但在1年保修期内，出现非人为性能故障，可免费维修。同一故障维修2次仍不能正常使用的，可凭维修记录换货。

## 退款流程与时效

### 退款到账时间
| 支付方式 | 到账时间 |
|---------|---------|
| 支付宝 | 1-3个工作日 |
| 微信支付 | 1-3个工作日 |
| 银行卡 | 3-7个工作日 |
| 花呗 | 额度即时恢复，账单1-3个工作日调整 |
| 京东白条 | 额度即时恢复 |

### 退货退款流程
1. 在购买平台提交退货申请
2. 审核通过后安排快递上门取件
3. 商家签收并检测（约1-3天）
4. 检测通过后发起退款
5. 按原支付方式退回

## 特殊场景

### 赠品怎么处理？
退货时需将赠品一同寄回，如赠品已使用或丢失，将按赠品吊牌价从退款中扣除。

### 拆封了还能退吗？
手机作为特殊电子产品，拆封后如未激活（未联网、未插SIM卡）仍可退；激活后无质量问题不予退货。建议先体验展示机再做购买决定。

### 购机后降价能退差价吗？
大部分平台和品牌不支持价保（价格保护）。京东自营部分手机支持7-30天价保期，下单时可注意查看"价保"标识。拼多多百亿补贴不保价。
