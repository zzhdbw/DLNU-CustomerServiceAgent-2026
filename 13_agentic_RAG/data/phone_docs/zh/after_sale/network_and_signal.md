# 网络与信号问题

## 信号问题排查

### 手机信号差怎么办？
1. 尝试开关飞行模式（重新搜索信号）
2. 取下手机壳（金属/磁吸壳会屏蔽信号）
3. 移动到开阔区域测试
4. 检查是否开启了省电模式（省电模式会降级网络）
5. 对比其他运营商卡在同一位置的信号强度
6. 更新系统基带版本（系统更新中通常含基带优化）
7. 如果仅某个位置信号差，可能是运营商基站覆盖问题

### 华为手机信号优势
华为在通信领域有深厚积累，Mate 60 Pro搭载灵犀通信技术，通过三路天线信号增强技术，在弱信号场景（电梯、地下车库、高铁）下比同位置的其他品牌手机信号强30%-50%。

### 小米手机信号表现
小米14采用全新天线架构设计，支持4×4 MIMO天线，信号表现稳定。相比华为在弱信号场景下稍弱，但在城市环境中使用无差别感受。

## 5G vs 4G

### 要不要开5G？
- 5G优势：下载大文件快（游戏/电影）、直播低延迟
- 5G劣势：耗电比4G高约20%、部分区域信号不稳定
- 建议：日常使用关闭5G用4G更省电，需要下载大文件时再开启5G
- 设置路径：设置→移动网络→网络类型选择→5G开关

### 华为手机为什么不支持5G？
华为Mate 60 Pro和后续机型搭载麒麟9000S芯片，网络制式显示为4G。华为因外部制裁原因，目前发布的手机暂不支持5G网络。但华为通过灵犀通信技术和卫星通信功能弥补了通信体验上的短板。日常使用4G+网络速度足够满足视频、社交、支付等需求。

## Wi-Fi相关问题

### Wi-Fi连不上怎么办？
1. 重启路由器（拔电30秒再插上）
2. 手机"忽略网络"后重新输入密码连接
3. 检查路由器是否限制了连接设备数
4. 尝试重启手机
5. 进入设置→WLAN→高级设置→重置WLAN配置
6. 检查系统是否有可用更新

### Wi-Fi经常断连怎么解决？
1. 关闭WLAN+（智能切换Wi-Fi/移动网络）功能
2. 在Wi-Fi设置中关闭"自动切换网络"选项
3. 固定Wi-Fi频段为2.4GHz（穿墙能力更强）或5GHz（速度更快）
4. 检查路由器是否过热，放置在通风处
5. 尝试更新路由器固件

## 蓝牙连接问题

### 蓝牙耳机连接不上？
1. 确保耳机处于配对模式（指示灯闪烁）
2. 手机蓝牙列表中删除旧记录后重新配对
3. 检查耳机是否已连接其他设备（先断开其他设备连接）
4. 重启手机和耳机
5. 重置耳机（各品牌重置方式不同，参考耳机说明书）

### 车载蓝牙连接不稳定？
1. 删除手机和车机中的蓝牙配对记录
2. 重新配对，确保连接时手机和车机靠近
3. 部分车型支持蓝牙优先连接，在车机设置中调整优先级
4. 更新车机系统固件
